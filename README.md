# 1.天马是什么？

```
一个开源的，可以部署到树莓派等单片机上的，机器人控制程序，或许也可以叫系统？
```

# 2.天马想实现的？

```
1.可以通过易经做一些决策和预测
2.包括不仅限于PID等控制算法
3.实现基于人工肌肉的灵巧手和四足步行算法
4.实现多个机器人协同工作（意思是，将这个程序部署在多个机器人后，机器人间可以分割,并且完成任务。例如对某个草莓园进行收获。或者打扫家庭卫生,又或者帮忙遛狗？）
5.一套UI系统，可以用浏览器或者平板电脑控制此机器人
6.一套基于slam的视觉定位系统
7.一套语音控制系统，可以通过语音控制（类似于siri）
8.集成一套或者多套专家系统，比如说照顾草莓的专家？照顾猕猴桃的专家？纠正拳击动作的专业？或者拳击陪练？通过人脸表情来判断对方是否愿意和你谈恋爱？或者公司间对业务谈判时，判断对方的意图？
9.一个类似于飞机的黑盒子系统（机器人好的和坏的地方都需要记录，以及改善）
```

# 3.实现顺序

```
1.slam(因为实现一些实际功能的时候，首先要明白自己在哪里)
2.灵巧手和四足步行算法(实现简单的移动和抓握等脚本)
3.ui控制，可以下达基本指令
4.打扫卫生专家系统，证明下这个机器人不是个废物
5.黑盒子，记录数据并分析
6.易经做决策
7.多机器人合作
8.语音控制
```

# 2. What does Tianma want to achieve?

```
1. You can make some decisions and forecasts through the Book of Changes
2. Including not limited to PID and other control algorithms
3. Realize the dexterous hand and quadruped walking algorithm based on artificial muscles
4. Realize the collaborative work of multiple robots (meaning, after deploying this program on multiple robots, the robots can be divided and complete tasks. For example, harvesting a certain strawberry field. Or cleaning the house, or helping to walk the dog ?)
5. A set of UI system, you can use a browser or a tablet to control the robot
6. A set of slam-based visual positioning system
7. A set of voice control system, which can be controlled by voice (similar to siri)
8. Integrate one or more sets of expert systems, such as experts who take care of strawberries? Expert in caring for kiwis? A major in correcting boxing moves? Or boxing sparring? Judging whether the other party is willing to fall in love with you by facial expressions? Or judge the intention of the other party when negotiating business between companies?
```
